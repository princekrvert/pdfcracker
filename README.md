<h2> Installation cmd </h2>

#### currently only available for android

* `apt update && apt upgrade`
* `pkg install git`
* `git clone https://github.com/princekrvert/pdfcracker`
* `cd pdfcracker`
* `chmod +x *`
* `./pdfcracker <pdffile> <wordlist>`
